# 草莓单目三维姿态与尺寸估计系统总体方案设计

## 1. 系统总体数据流向 (Pipeline)

系统遵循从“像素空间”到“物理空间”的转换逻辑，完整的数据流向如下：

1.  **图像预处理阶段**：输入单目 RGB 图像，将其统一缩放至 $800 \times 600$ 像素，并进行归一化处理（ToTensor），转化为模型可接收的四维张量 $[1, 3, 600, 800]$。
2.  **特征提取与参数回归阶段**：通过深度卷积神经网络（Darknet 骨干网络）提取图像特征。网络输出一个 $22 \times 18 \times 25$ 的多通道特征图，其中包含了每个网格单元（Grid Cell）对草莓目标的置信度、2D 投影顶点偏移量以及 3D 物理尺寸的预测值。
3.  **目标过滤与参数解码阶段**：
    *   应用 Sigmoid 激活函数处理置信度逻辑值，筛选出高于阈值（如 0.60）的候选网格。
    *   解析 22 维预测向量，获取草莓中心的 2D 坐标、8 个立方体顶点的投影偏移量，以及三维长、宽、高参数。
    *   进行物理尺寸合理性校验（如 $10mm \sim 150mm$ 范围约束）。
4.  **几何约束求解 (PnP) 阶段**：利用预测的 2D 投影顶点坐标与基于预测尺寸构建的 3D 模型坐标，结合相机内参矩阵 $K$，通过 EPnP 算法求解透视 n 点问题（Perspective-n-Point），获取草莓相对于相机的旋转向量（rvec）和平移向量（tvec，即 $x, y, z$ 坐标）。
5.  **后处理与输出阶段**：通过基于欧氏距离的非极大值抑制（NMS）去除冗余检测框，最终输出草莓的三维姿态坐标、物理尺寸参数及可视化 3D 边界框。

---

## 2. 总体方案学术描述 (论文可用)

本系统设计了一种基于深度学习与几何约束相结合的单目三维姿态估计框架。系统核心架构采用**多任务并行回归策略**，将目标检测与三维几何参数估计集成于统一的网络输出中。

在算法流程上，系统首先利用卷积神经网络完成从图像像素到几何特征的非线性映射。不同于传统的二阶段方法，本方案通过**网格化预测（Grid-based Prediction）**机制，在单次前向传播中同时回归出草莓目标的 2D 关键点偏移量与 3D 物理尺寸。随后，系统引入**几何一致性约束**，将 2D 图像特征与 3D 物理模型关联。通过求解 PnP 问题的闭式解，系统能够从单帧二维图像中恢复出丢失的深度信息。此外，系统通过引入**物理先验过滤**（如尺寸范围约束）与**空间冗余抑制**（NMS），确保了在复杂背景及遮挡环境下，输出的草莓姿态参数（$x, y, z$）与尺寸参数（$l, w, h$）具备高置信度与物理鲁棒性。

## 3. 系统架构图逻辑结构推荐

建议在画系统架构图时，采用以下**模块化分层结构**：

### **模块 A：图像获取与预处理模块**
*   **功能**：负责原始图像读取、色彩空间转换（RGB/BGR）、重采样（Resize）以及数据张量化。
*   **逻辑位置**：系统的起始端。

### **模块 B：深度特征提取回归模块 (CNN Engine)**
*   **功能**：
    *   **Backbone (Darknet)**：多尺度特征提取。
    *   **Prediction Head**：并行输出置信度图、2D 关键点偏移图、3D 尺寸回归图。
*   **逻辑位置**：系统的核心计算单元。

### **模块 C：几何参数解码模块 (Parameter Decoder)**
*   **功能**：应用 Sigmoid 激活、网格坐标转换，将神经网络输出的相对偏移量还原为图像坐标系下的绝对像素坐标。
*   **逻辑位置**：神经网络输出后的第一个处理层。

### **模块 D：姿态与尺寸解析模块 (Pose & Size Solver)**
*   **功能**：
    *   **3D 模型重建**：根据回归的 $l, w, h$ 构建 3D 参考点。
    *   **EPnP 求解器**：结合相机内参执行 2D-3D 匹配运算。
    *   **深度检验**：对求解结果进行 $z$ 轴距离合法性校验。
*   **逻辑位置**：从像素到物理参数转化的关键环节。

### **模块 E：结果优化与可视化输出模块**
*   **功能**：执行 Euclidean-NMS 过滤重复目标，并在原图上绘制 3D 投影边界框。
*   **逻辑位置**：系统的输出终端。
